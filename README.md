# PlatinumRx Assignment README

## Overview

This project demonstrates basic data analysis skills using SQL, spreadsheets, and Python. It includes database creation, data querying, spreadsheet-based analysis, and simple Python programs.

## Project Structure

Data_Analyst_Assignment/

* SQL/

  * 01_Hotel_Schema_Setup.sql
  * 02_Hotel_Queries.sql
  * 03_Clinic_Schema_Setup.sql
  * 04_Clinic_Queries.sql

* Spreadsheets/

  * Ticket_Analysis.xlsx

* Python/

  * 01_Time_Converter.py
  * 02_Remove_Duplicates.py

* README.md

## Description

### SQL

Contains scripts for creating tables and inserting data for Hotel and Clinic systems. It also includes queries to analyze bookings, billing, revenue, and other metrics.

### Spreadsheets

Includes an Excel file used for ticket and feedback analysis. Functions like VLOOKUP, date comparison, and counting conditions are used.

### Python

Includes two scripts:

* Time Converter: Converts minutes into hours and remaining minutes.
* Remove Duplicates: Removes duplicate characters from a string while maintaining order.

## Tools Used

* SQL (MySQL / MySQL Workbench)
